# KoExplorer 仕様書

バージョン: 1.1.0  
最終更新: 2026-04-13

---

## 概要

KoExplorerは、Windowsエクスプローラーのような使い慣れたインターフェースで画像を快適に閲覧・管理できる、Windows向け高速イメージビューアーです。

- **プラットフォーム**: Windows 10 (1809以降) / Windows 11
- **フレームワーク**: WPF (.NET 9.0)
- **アーキテクチャ**: MVVM (CommunityToolkit.Mvvm)

## 機能仕様

### 1. ナビゲーション

#### 1.1 フォルダツリー（左ペイン）
- クイックアクセス（マイピクチャ、デスクトップ）とPCドライブを表示
- フォルダ展開時に子フォルダを遅延読み込み
- 隠しフォルダ・システムフォルダは非表示
- フォルダ選択で右ペインのファイル一覧を更新
- F5キーまたはリフレッシュボタンでツリーを再構築
- 幅はスプリッターで調整可能（最小150px）

#### 1.2 アドレスバー
- 現在のフォルダパスを表示・編集
- Enterキーまたは「→」ボタンで入力パスに移動
- Escキーで入力をキャンセルして元のパスに戻す
- フォーカス時に全選択
- 存在しないパスを入力した場合はエラーをステータスバーに表示
- フォルダ移動時にツリーも自動展開

#### 1.3 画像ナビゲーション
- 次の画像: →キー / Nキー / ツールバーボタン
- 前の画像: ←キー / Pキー / ツールバーボタン
- 最初の画像: Homeキー
- 最後の画像: Endキー
- マウスホイール（Ctrl無し）で画像切り替え

#### 1.4 起動時の復元
- 前回開いたフォルダを自動的に開く
- フォルダツリーも前回のフォルダまで自動展開

---

### 2. ファイル一覧表示

#### 2.1 詳細表示モード
- DataGridによるテーブル形式表示
- カラム構成:
  - チェックボックス（ヘッダーで全選択/全解除）
  - ファイル名（フォルダ=オレンジアイコン、ファイル=ブルーアイコン）
  - 更新日時（yyyy/MM/dd HH:mm）
  - 種類（ファイルタイプ）
  - サイズ（フォルダは非表示、右寄せ）
- 先頭に「..」（親フォルダ）を表示
- ダブルクリックでフォルダを開く

#### 2.2 サムネイル表示モード
- WrapPanelによるグリッド形式表示
- サムネイルサイズをスライダーで64〜256pxに調整
- 各サムネイルの左上にチェックボックスを表示
- ダブルクリックでフォルダを開く
- プレビューエリアとサムネイルパネルの2分割表示
- プレビュー非表示時はサムネイルパネルが全画面表示

#### 2.3 ファイル一覧の共通仕様
- フォルダとファイルを混在表示（フォルダが先頭）
- 並び替え: ファイル名/更新日時/ファイルサイズ（昇順/降順）
- 右クリックでコンテキストメニューを表示

---

### 3. プレビュー機能

#### 3.1 画像表示
- ScrollViewerによるスクロール対応
- 高品質レンダリング（BitmapScalingMode.HighQuality）
- 左上に画像情報オーバーレイ表示（ファイル名、解像度、サイズ、形式）
- 読み込み失敗時にエラーメッセージを表示

#### 3.2 ズーム機能
| モード | 操作 | 説明 |
|--------|------|------|
| ウィンドウにフィット | Fキー / Ctrl+F | 画像全体がウィンドウに収まるよう表示 |
| 幅に合わせる | Wキー | 画像幅をウィンドウ幅に合わせる |
| 高さに合わせる | Hキー | 画像高さをウィンドウ高さに合わせる |
| 実寸表示 | 1キー / Ctrl+0 | 100%表示 |
| 拡大 | Ctrl++ | 1.25倍ずつ拡大（最大16倍） |
| 縮小 | Ctrl+- | 1.25倍ずつ縮小（最小0.1倍） |
| マウスズーム | Ctrl+ホイール | ホイール操作でズーム |

- ウィンドウリサイズ時にフィット系モードは自動再計算

#### 3.3 パン操作
- Ctrl+ドラッグで画像をスクロール

#### 3.4 プレビューからのD&D
- 通常ドラッグでファイルD&D開始（左ペインツリーや右ペイン内フォルダへ移動）

#### 3.5 背景色
- 白 / グレー（デフォルト）/ 黒 から選択

#### 3.6 フルスクリーン
- F11キーまたはメニューで切り替え
- 解除時に元のウィンドウ状態を復元

---

### 4. ファイル操作

#### 4.1 削除
- 優先順位: チェック済み > 選択中 > 現在表示中の画像
- ゴミ箱に移動（完全削除ではない）
- 削除前に確認ダイアログを表示
- 複数ファイルの場合はファイル名一覧を表示（最大5件）

#### 4.2 移動
- **D&Dによる移動**:
  - 右ペインのファイルを左ペインのツリーにドロップ
  - 右ペインのファイルを同じ一覧内のフォルダにドロップ
  - プレビュー画像を左ペインのツリーにドロップ
  - ドロップ先フォルダをハイライト表示
- **チェック済み移動**: ツールバーの移動ボタンでフォルダ選択ダイアログを表示
- 移動前に確認ダイアログを表示

#### 4.3 名前変更
- 右クリックメニュー「名前の変更」
- 専用ダイアログで入力（拡張子の前まで自動選択）
- Enterで確定、Escでキャンセル
- 1ファイル/フォルダのみ対応

#### 4.4 コピー
- 画像をクリップボードにコピー: Ctrl+C
- ファイルパスをコピー: Ctrl+Shift+C
- 切り取り（Windows標準形式）: 右クリックメニュー

#### 4.5 その他
- エクスプローラーで開く: Ctrl+E
- 既定のアプリで開く: Ctrl+O（メニューはCtrl+O）

---

### 5. 複数ファイル選択

#### 5.1 チェックボックス選択
- 各行/サムネイルのチェックボックスをクリックして選択
- ヘッダーのチェックボックスで全選択/全解除
- チェック済みアイテムに対して削除・移動を一括実行

#### 5.2 行選択（DataGrid）
- Ctrlクリックで個別追加/解除
- Shiftクリックで範囲選択
- 右クリック時、選択済みアイテムの選択を維持

---

### 6. コンテキストメニュー（右クリック）

詳細表示・サムネイル表示の両方で共通:

| メニュー項目 | 動作 |
|-------------|------|
| 開く | フォルダを開く / 既定のアプリで開く |
| エクスプローラーで開く | Windowsエクスプローラーで表示 |
| 切り取り | Windows標準形式でクリップボードに切り取り |
| コピー | 画像をクリップボードにコピー |
| パスをコピー | ファイルパスをクリップボードにコピー |
| 名前の変更 | 名前変更ダイアログを表示 |
| 削除 | ゴミ箱に移動 |

---

### 7. 対応画像フォーマット

| 形式 | 拡張子 |

| WebP | .webp |
| BMP | .bmp |
| GIF | .gif |

### 9. 設定

設定ファイルの保存場所: `%APPDATA%\KoExplorer\settings.json`

| 設定項目 | デフォルト値 | 説明 |
|---------|------------|------|
| ThumbnailSize | 128 | サムネイルサイズ（px） |
| ThumbnailCacheSize | 500 | サムネイルキャッシュ（MB） |
| BackgroundColor | Gray | プレビュー背景色 |
| ShowFolderTree | true | フォルダツリー表示 |
| ShowThumbnailPanel | true | サムネイルパネル表示 |
| ShowImageInfo | true | 画像情報オーバーレイ表示 |
| LastOpenedFolder | "" | 前回開いたフォルダ |
| DefaultZoomMode | Fit | デフォルトズームモード |
