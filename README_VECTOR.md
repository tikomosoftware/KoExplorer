# KoExplorer v1.0.0

Windows向けエクスプローラー風イメージビューアー

## 概要

KoExplorerは、Windowsエクスプローラーのような使い慣れたインターフェースで画像を快適に閲覧できる、高速イメージビューアーです。

## 主な機能

### 🖼️ 多様な画像フォーマット対応
- **JPG/JPEG** (.jpg, .jpeg)
- **PNG** (.png)
- **WebP** (.webp)
- **BMP** (.bmp)
- **GIF** (.gif)
- **TIFF** (.tiff, .tif)
- **ICO** (.ico)
- **SVG** (.svg)

### ⚡ 高速表示
- サムネイルキャッシュによる高速プレビュー
- 大量の画像ファイルもスムーズに閲覧
- 遅延読み込みによる快適な起動

### 🎯 使いやすいインターフェース
- エクスプローラー風の3ペイン構成
- フォルダツリーで素早くフォルダ移動
- 詳細表示とサムネイル表示の切り替え
- 複数ファイルの一括削除

### 🔍 豊富なズーム機能
- 実寸表示、ウィンドウにフィット、幅に合わせる、高度に合わせる
- 自由な拡大・縮小

## 🚀 インストール・使用方法

### 📦 パッケージ内容
本パッケージは **.NET同梱版** です。
.NET Runtimeが含まれているため、別途インストール不要で、すぐに使い始められます。

### インストール手順
1. ダウンロードしたZIPファイルを任意の場所に解凍
2. `KoExplorer.exe`を実行

## 使い方

### 基本操作
1. アプリを起動
2. 左ペインのフォルダツリーからフォルダを選択、または「フォルダを開く」ボタンをクリック
3. 右上ペインで画像ファイルを選択
4. 右下ペインでプレビュー表示

### キーボードショートカット
- `→` / `N`: 次の画像
- `←` / `P`: 前の画像
- `Ctrl` + `+`/`-`: 拡大/縮小
- `F11`: フルスクリーン切り替え
- `Delete`: ゴミ箱に移動

## 🔧 トラブルシューティング

### アプリが起動しない
- Windows 10の場合、Version 1809以降であることを確認してください

### 画像が表示されない
- 対応フォーマットであることを確認してください
- ファイルが破損していないか確認してください
