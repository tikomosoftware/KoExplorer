# KoExplorer 実装メモ

## プロジェクト概要

KoExplorerは、Windowsエクスプローラーのような使いやすいインターフェースを持つ画像ビューアーです。
WPF + .NET 9.0で構築され、MVVMアーキテクチャを採用しています。

## 現在の実装状況

### ✅ 完了した機能

#### 1. プロジェクト構造
- WPF .NET 9.0 プロジェクト
- MVVM アーキテクチャ
- 依存性注入の設定（Microsoft.Extensions.DependencyInjection）

#### 2. UI実装
- **メインウィンドウ**: エクスプローラー風3ペインレイアウト
  - 左ペイン: フォルダツリー
  - 中央ペイン: プレビューエリア
  - 右ペイン: サムネイルパネル
- **メニューバー**: ファイル、表示、ヘルプ
- **ツールバー**: よく使う機能へのクイックアクセス
- **ステータスバー**: 画像情報とズーム率の表示

#### 3. 画像表示機能
- 多様な画像フォーマット対応（JPG, PNG, WebP, BMP, GIF, TIFF, ICO, SVG）
- サムネイル生成とキャッシュ
- 画像ナビゲーション（次/前/最初/最後）
- ズーム機能（拡大/縮小/実寸/フィット）
- マウスドラッグでパン操作
- マウスホイールでズーム/画像切り替え

#### 4. ファイル操作
- 画像をクリップボードにコピー
- ファイルパスをコピー
- ゴミ箱に移動
- エクスプローラーで開く
- 既定のアプリで開く

#### 5. 表示機能
- 詳細表示/サムネイル表示の切り替え
- 並び替え（名前/日時/サイズ、昇順/降順）
- 背景色の変更（白/グレー/黒/チェッカーボード）
- フルスクリーンモード
- パネルの表示/非表示切り替え

#### 6. サービス層
- **ImageLoaderService**: 画像読み込みと情報取得
- **ThumbnailService**: サムネイル生成とキャッシュ管理
- **FileSystemService**: ファイルシステム操作とフォルダツリー構築
- **SettingsService**: アプリケーション設定の保存/読み込み
- **LogService**: デバッグログ（現在は無効化）

#### 7. パフォーマンス最適化
- フォルダツリーの遅延読み込み
- 非同期初期化でUIスレッドをブロックしない
- サムネイルの非同期生成
- ログ出力の無効化による高速起動

### 🚧 既知の問題と制限事項

#### 1. フォルダツリー
- フォルダ選択時の自動同期機能は削除済み（安定性のため）
- ユーザーが手動でツリーを展開する必要がある

#### 2. ズーム機能
- ウィンドウにフィット/幅に合わせる/高さに合わせるは簡易実装
- マウスカーソル位置を中心にしたズームは未実装

#### 3. SVG対応
- 基本的な表示は可能だが、複雑なSVGは正しく表示されない可能性

#### 4. アイコンリソース
- 一部のアイコン画像が存在しないため、実行時に表示されない

## アーキテクチャ

### MVVMパターン
- **Model**: データモデル（Models/）
- **View**: XAML UI（MainWindow.xaml, Styles/）
- **ViewModel**: ビジネスロジック（ViewModels/MainViewModel.cs）

### 依存性注入
- サービスはインターフェースを通じて注入
- テストやメンテナンスが容易

### 非同期処理
- 画像読み込み、サムネイル生成は非同期
- UIスレッドをブロックしない

## 開発ガイドライン

### コーディング規約
- C# 12の最新機能を活用
- nullable参照型を有効化
- XMLドキュメントコメントを記述

### パフォーマンス
- 大量の画像でも快適に動作するよう最適化
- メモリ使用量に注意
- 非同期処理を積極的に活用

### エラーハンドリング
- ユーザーに分かりやすいエラーメッセージ
- ログに詳細情報を記録（デバッグ時）
- アプリケーションがクラッシュしないよう適切に処理

## トラブルシューティング

### 起動が遅い
- ログ出力が有効になっていないか確認
- フォルダツリーの遅延読み込みが機能しているか確認

### 画像が表示されない
- 対応フォーマットか確認
- ファイルが破損していないか確認
- エラーメッセージを確認

### メモリ使用量が多い
- サムネイルキャッシュのサイズを確認
- 大量の画像を開いていないか確認
