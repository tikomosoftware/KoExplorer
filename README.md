# KoExplorer v1.0.0

Windows向けエクスプローラー風イメージビューアー

![KoExplorer](https://img.shields.io/badge/version-1.0.0-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)
![.NET](https://img.shields.io/badge/.NET-9.0-purple)

## 概要

KoExplorerは、Windowsエクスプローラーのような使い慣れたインターフェースで画像を快適に閲覧できる、高速イメージビューアーです。

## 主な機能

### 🖼️ 多様な画像フォーマット対応
- **JPG/JPEG** (.jpg, .jpeg)
- **PNG** (.png)
- **WebP** (.webp)
- **BMP** (.bmp)
- **GIF** (.gif)
- **TIFF** (.tiff, .tif)
- **ICO** (.ico)
- **SVG** (.svg)

### ⚡ 高速表示
- サムネイルキャッシュによる高速プレビュー
- 大量の画像ファイルもスムーズに閲覧
- 遅延読み込みによる快適な起動

### 🎯 使いやすいインターフェース
- エクスプローラー風の3ペイン構成
- フォルダツリーで素早くフォルダ移動
- 詳細表示とサムネイル表示の切り替え
- 複数ファイルの一括削除

### 🔍 豊富なズーム機能
- 実寸表示
- ウィンドウにフィット
- 幅に合わせる
- 高さに合わせる
- 自由な拡大・縮小

### ⌨️ キーボードショートカット
効率的な操作のための豊富なショートカットキー（詳細は下記参照）

## システム要件

- **OS**: Windows 10 (Version 1809以降) または Windows 11
- **.NET**: .NET 9.0 Runtime
- **推奨メモリ**: 4GB以上
- **推奨ストレージ**: SSD推奨（高速なサムネイル生成のため）

## インストール

[Releases](../../releases)ページから、環境に応じたパッケージをダウンロードしてください。

### .NET同梱版（推奨）
**ファイル名**: `KoExplorer-v1.0.0-win-x64.zip`

- .NET Runtimeが含まれているため、別途インストール不要
- ファイルサイズが大きい（約70MB）
- すぐに使い始められる

**インストール手順**:
1. ZIPファイルをダウンロード
2. 任意の場所に解凍
3. `KoExplorer.exe`を実行

### .NET非同梱版（軽量版）
**ファイル名**: `KoExplorer-v1.0.0-framework-dependent.zip`

- .NET Runtimeが別途必要
- ファイルサイズが小さい（約1MB）
- 既に.NET 9.0がインストールされている場合に最適

**インストール手順**:
1. ZIPファイルをダウンロード
2. 任意の場所に解凍
3. .NET 9.0 Runtimeがインストールされていない場合は、[こちら](https://dotnet.microsoft.com/download/dotnet/9.0)からダウンロードしてインストール
4. `KoExplorer.exe`を実行

## 使い方

### 基本操作
1. アプリを起動
2. 左ペインのフォルダツリーからフォルダを選択、または「フォルダを開く」ボタンをクリック
3. 右上ペインで画像ファイルを選択
4. 右下ペインでプレビュー表示

### キーボードショートカット

#### ナビゲーション
| キー | 機能 |
|------|------|
| `→` または `N` | 次の画像 |
| `←` または `P` | 前の画像 |
| `Home` | 最初の画像 |
| `End` | 最後の画像 |

#### ズーム
| キー | 機能 |
|------|------|
| `Ctrl` + `+` | 拡大 |
| `Ctrl` + `-` | 縮小 |
| `Ctrl` + `0` または `1` | 実寸表示 |
| `F` または `Ctrl` + `F` | ウィンドウにフィット |
| `W` | 幅に合わせる |
| `H` | 高さに合わせる |
| `マウスホイール` + `Ctrl` | ズーム |

#### 表示
| キー | 機能 |
|------|------|
| `F11` | フルスクリーン切り替え |
| `F3` | サムネイルパネル表示切り替え |

#### ファイル操作
| キー | 機能 |
|------|------|
| `Ctrl` + `C` | 画像をクリップボードにコピー |
| `Ctrl` + `Shift` + `C` | パスをコピー |
| `Delete` | ゴミ箱に移動 |
| `Ctrl` + `E` | エクスプローラーで開く |
| `Ctrl` + `O` | 既定のアプリで開く |

### マウス操作
- **画像のドラッグ**: パン（移動）
- **マウスホイール**: 画像切り替え
- **Ctrl + マウスホイール**: ズーム
- **ダブルクリック**: フォルダを開く

## トラブルシューティング

### アプリが起動しない
- .NET 9.0 Runtimeがインストールされているか確認してください
- Windows 10の場合、Version 1809以降であることを確認してください

### 画像が表示されない
- 対応フォーマットであることを確認してください
- ファイルが破損していないか確認してください

### サムネイルの生成が遅い
- 初回表示時はサムネイル生成に時間がかかる場合があります
- SSDの使用を推奨します
